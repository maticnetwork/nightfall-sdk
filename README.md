## Setup

Clone the repo:

```bash
git clone git@github.com:maticnetwork/nightfall-sdk.git
cd nightfall-sdk
```

Set Node.js vs to 16.15.1:

```bash
nvm use
```

Install requirements:

```
npm install
```

## Getting started

### How to make a deposit
